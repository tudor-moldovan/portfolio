Edit Positions

JSON array · stored in Vercel KV · takes effect on next home load
Each position needs: symbol, quoteSym, broker, asset, currency (USD|EUR|RON), units, avgCost. Optional: targetBuy, targetSell, stopLoss, isLump. For duplicates (same ticker on two brokers), suffix the symbol (e.g. BTC-R) but set quoteSym: "BTC". For index-tracker accounts where only RON invested is known, set isLump: true and use avgCost as total RON and units: 1.
